Surgimento da internet

(Voltar)

Criada em 1989 por Tim Berners-Lee, que viu a oportunidade de unir hipertexto com TCP/IP e criar a World Wide Web (WWW)

Como funciona o HTML

(Voltar)

O HTML é um arquivo de texto puro com a extensão .html, armazenado em um servidor.

Quando um usuário digita o endereço de um site no navegador, o navegador solicita uma cópia do documento HTML ao servidor.

O navegador interpreta o código HTML, de cima para baixo, e exibe o conteúdo de acordo com as instruções das tags e atributos.

O que são Tags

(Voltar)

As tags são os blocos de construção do HTML, que indicam ao navegador como apresentar as informações. Por exemplo, as tags diferenciam títulos de parágrafos, links e menus.

No HTML, elas são formadas por três tags básicas: table, para delimitar a tabela; tr, para indicar as linhas; e td para formar as colunas.

O que são atributos

(Voltar)

Os atributos são uma parte crucial de um elemento HTML, que possuem um nome e um valor. O nome identifica a informação adicional que o usuário deseja acrescentar, e o valor de atributo fornece mais especificações.
